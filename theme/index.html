Pages

How we load the CSS, once received

To cater for as many devices as possible, we start with a feature phone stylesheet that is as small as possible, but covers our bases. Once that is done, we add the enhanced stylesheet below that (using feature detection). The enhanced stylesheet then inherits from the feature phone styling.

Please note that both your feature and enhanced stylesheets will inherit from the default theme.

How to go about theming the pages

  • We've included the feature and enhanced stylesheets into the head of every page below in exactly the same way it will be done once integrated.
  • They are located at /pages/static/authentication_service/css/theme.feature.css and /pages/static/authentication_service/css/theme.enhanced.css
  • You can now open those up in your favourite editor and start creating your own theme.

Both these template files look exactly the same, with the same classes. You'll only use some of them for feature styling. Please make sure to remove the unused declarations.

Feel free to add whatever you need to either of these files to get to your own perfectly customized theme.

Directory naming & referencing

Please make sure to name your images folder images and your fonts folder fonts.

Get started with your theme

Use the links in the sidebar to navigate to the various pages that you will need to cater for.

Sending your theme

Please remember to send along any images and fonts that you reference in your CSS files.
