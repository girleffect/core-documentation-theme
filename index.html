Theming

This theme pack is broken up into 2 sections, components and pages.

Components

Components are all the various building blocks that are used,
from typography to form error messages.

Pages

Pages are the specific pages that are used in the core authentication service,
and they show how the components get used. Upon changing your theme,
you will be able to view those changes on every individual page.

Template

You can find a template stylesheet for feature phone here, and for enhanced phones here.
These already have all the class names you need.

You can just edit the templates and start viewing your changes on the various pages.

We cater for as many devices as possible, so please provide both a feature and an enhanced stylesheet.
